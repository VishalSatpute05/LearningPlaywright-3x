# IQ Nodes — Concept Explainer

```
I want to understand [concept name].
Create an IQ file for it in iq.nodes/ using the same table format you used for source code vs bytecode vs binary code.
```

## What I'll do

I'll create a markdown file in this folder named `[ConceptName]_IQ.md` with:

- A **comparison table** breaking down the concept
- A **real-world example** from this project (if applicable)
- A **visual flow** or diagram in ASCII
- Clear, beginner-friendly explanations

## File naming convention

- `Source_code_byteCode_Binary_IQ.md`
- `[ConceptName]_IQ.md`
